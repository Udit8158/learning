# Starting point: competent developer, complete Hugging Face beginner

Udit is an experienced software developer but a complete beginner to Hugging Face and to
AI/ML development in general. He has not yet run a model himself.

Implications:
- Skip general programming instruction; spend working memory on AI-specific concepts.
- Floor for teaching is "what is Hugging Face and how do I run one model" — not weights,
  architectures, or training.
- Mission is creative/automation (images, video, scripts), so every concept should ladder
  toward running real generative models, not NLP-classification toy examples.
