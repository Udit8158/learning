# ComfyUI & Local Image-Gen Resources

## Knowledge

- [ComfyUI Official Documentation — docs.comfy.org](https://docs.comfy.org/)
  The canonical reference. Use for: core concepts, node behaviour, official workflow templates, install/update.
- [How does Stable Diffusion work? — Stable Diffusion Art](https://stable-diffusion-art.com/how-stable-diffusion-work/)
  Best beginner-friendly explainer of latent diffusion (VAE, U-Net, CLIP, sampler). Use for: the "why" behind the pipeline.
- [Beginner's Guide to ComfyUI — Stable Diffusion Art](https://stable-diffusion-art.com/comfyui/)
  Hands-on walkthrough of the default text-to-image graph. Use for: mapping concepts to actual nodes.
- [Latent Diffusion Models Explained — Louis Bouchard](https://www.louisbouchard.ai/latent-diffusion-models/)
  Clear, slightly deeper conceptual treatment. Use for: reinforcing the latent-space idea.
- [Install ComfyUI on Mac — ComfyUI Wiki](https://comfyui-wiki.com/en/install/install-comfyui/install-comfyui-on-mac)
  Apple Silicon specifics: MPS backend, `--force-fp16`. Use for: Mac performance tuning, flags, troubleshooting.
- [Consistent Characters in ComfyUI (IPAdapter + ControlNet), 2025 — Medium](https://tgecrypto365.medium.com/how-to-create-consistent-characters-comfyui-the-2025-step-by-step-workflow-ipadapter-76edbfca0baf)
  Practical character-consistency workflow. Use for: later manga-panel lessons (not yet — keep for ZPD).

- [r/comfyui](https://reddit.com/r/comfyui)
  Active, workflow-sharing subreddit. Use for: troubleshooting graphs, discovering nodes, manga/comic techniques.
- [r/StableDiffusion](https://reddit.com/r/StableDiffusion)
  Largest local-image-gen community. Use for: model news, prompting wisdom, Mac-specific threads.
- [ComfyUI Matrix/Discord (linked from docs.comfy.org)](https://docs.comfy.org/)
  Direct line to power users and devs. Use for: real-time help on stuck workflows.
