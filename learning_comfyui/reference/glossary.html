Glossary

The shared vocabulary of the course. Every lesson uses these terms exactly as defined here.

The pipeline

TermMeaning
DiffusionGenerating an image by starting from pure noise and removing a little noise at a time until a picture emerges. The model was trained by watching images get noised, so it learns to reverse the process.
Latent spaceA compressed numeric representation of an image (~48× smaller than pixels). Diffusion happens here, not on pixels — that's why it fits on a 16 GB Mac.
LatentOne image's worth of data in latent space. Starts as noise, ends as the (still-compressed) finished image.
VAEVariational Auto-Encoder. The translator between pixels and latent space. Encodes pixels → latent, decodes latent → pixels.
U-NetThe denoising model — the "brain" that predicts what noise to remove at each step. The bulk of a checkpoint's size.
CLIPThe text encoder. Turns your prompt words into numbers (an embedding) the U-Net can be steered by.
ConditioningThe encoded prompt — the "instruction" fed to the sampler. There's a positive one (want) and a negative one (avoid).
Sampler / SchedulerThe algorithm that runs the denoising loop step by step (e.g. euler, dpmpp_2m). Lives in the KSampler node.
StepsHow many denoising iterations to run. More steps = more refinement, more time. ~20–30 is typical.
CFG scaleClassifier-Free Guidance. How hard to push toward the prompt. Low = loose/creative, high = literal/over-baked. ~6–8 typical.
SeedThe number that picks the starting noise. Same seed + same settings = same image. Your reproducibility knob.

Models & files

TermMeaning
CheckpointA trained model file bundling U-Net + CLIP + (usually) VAE. The style/knowledge of your generator. e.g. an anime checkpoint for manga.
SD1.5Stable Diffusion 1.5. Older, small, fast; native 512×512. Great first model on the M1.
SDXLStable Diffusion XL. Bigger, sharper; native 1024×1024. Runs on 16 GB M1 (~60–90s/image).
FluxA newer, larger model family. Excellent quality, heavy — slow locally; a cloud-GPU candidate later.
LoRAA small add-on file that nudges a checkpoint toward a specific style/character without retraining it. Key tool for consistent manga characters.

ComfyUI machinery

TermMeaning
NodeA box that does one job. Think of it as a pure function: typed inputs on the left, typed outputs on the right.
Edge / wireA connection carrying one typed value from a node's output to another's input. The data flow of your graph.
Workflow / graphThe whole network of connected nodes. Saved as JSON — also embedded inside PNGs you generate.
MPSMetal Performance Shaders — Apple's GPU backend. ComfyUI uses it automatically on your M1 (look for Using device: mps).
--force-fp16Launch flag that halves precision to save memory on Apple Silicon, with little quality loss. Recommended on the M1.
