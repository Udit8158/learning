# learning

My self-directed learning workspaces, built with Claude Code's `/teach` skill.
Each folder is a self-contained course: a mission, curated resources, interactive
HTML lessons, reference docs, and a running record of what I've learned.

## Courses

- **[learning_comfyui](./learning_comfyui)** — ComfyUI & local image-generation workflows
  on Apple Silicon, working toward turning stories into manga panels.
- **[learning_huggingface](./learning_huggingface)** — the Hugging Face ecosystem and
  AI-engineering foundations.

Open any `lessons/*.html` in a browser to take a lesson.
